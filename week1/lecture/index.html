How the Web Works

Request a domain name from Domain Name System (DNS) servers

DNS server returns the IP Address of a web server

Request files from the web server using its IP Address

Semantic Markup

Semantic means "meaning"

Semantic HTML is focused on giving your content meaning.

  • Pay attention to the intended meaning of each piece of your content.
  • Use HTML tags that describe your content appropriately.
  • Don't misuse tags to achieve a particular visual effect.

Why We Should Care

  • Improved Search Engine Optimization (SEO)
  • Improved Accessibility
  • Lighter code and faster pages
  • Easier code to maintain and style
